ambient chronomancy · public epistemic infrastructure
AGGREGATE SOURCE

An architecture for continuous coupling detection, running as a new mode inside the Chronomancy platform. A server-side 1 MHz quantum random number generator streams bits continuously. A three-detector ensemble (runs test · χ² byte-frequency · block- frequency · Fisher combine) watches it. When the stream does something statistically unusual, the user gets a ping.

This repo is the epistemic infrastructure. It holds the architecture, the pre-registrations, the calibration certificates, and the daily Merkle-sealed event log. The git log is the chain.
